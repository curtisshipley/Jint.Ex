JintSetTimeoutDemo

Overview

The sample shows how to implement setTimeOut() function in Jint and is my first step at building a layer around Jint, to execute JavaScript program in background in relation with a UI (Windows WinForm and iOS UI).

The purpose of class AsyncronousEngine is to * Wait script in a background thread * Allow communication with the UI and UI Controls * Execute __setTimeout__ call back
